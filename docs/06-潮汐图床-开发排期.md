# 潮汐图床开发排期

## 1. 第一阶段：基础闭环

目标：完成能注册、登录、上传、查看图片的最小闭环。

任务：

- Rust 后端项目初始化。
- Leptos 前端项目初始化。
- PostgreSQL 连接。
- 数据库迁移框架。
- 用户注册。
- 用户登录。
- 邮箱验证码。
- 基础上传。
- 本地存储。
- 用户图片列表。
- 图片详情。
- Markdown / HTML / 原图链接生成。

验收：

- 用户可以邮箱注册。
- 用户可以登录。
- 用户可以上传图片到本地存储。
- 用户可以查看自己的图片。
- 用户可以复制 Markdown / HTML / 原图链接。

## 2. 第二阶段：公益图床核心能力

目标：降低存储压力并建立资源边界。

任务：

- 用户组配额。
- 访客上传。
- 哈希去重。
- WebP 预览图。
- 回收站。
- 标签系统。

验收：

- 管理员可配置用户组配额。
- 访客上传可开关。
- 访客上传归属到公用账号。
- 重复图片不重复存储真实文件。
- 上传后生成 WebP 预览图。
- 删除图片进入回收站。
- 用户可为图片添加标签。

## 3. 第三阶段：多存储

目标：让存储器接管上传、预览和调用。

任务：

- StorageProvider 统一接口。
- Cloudflare R2。
- OneDrive。
- Oracle Object Storage S3 兼容模式。
- Oracle Object Storage OCI 原生模式。
- 存储健康检查。
- 存储失败重试。
- 默认存储策略。

验收：

- 管理员可添加 R2 存储。
- 管理员可添加 OneDrive 存储。
- 管理员可添加 Oracle S3 存储。
- 管理员可添加 Oracle OCI 原生存储。
- 图片原图和 WebP 预览图可写入指定存储。
- 图片访问 URL 由存储适配器生成。
- 随机图调用可返回原图或预览图。

## 4. 第四阶段：管理员后台

目标：让管理员能完整运营潮汐图床。

任务：

- 管理员仪表盘。
- 用户管理。
- 图片管理。
- 用户组管理。
- 标签管理。
- 存储管理。
- 系统设置。
- 主题设置。
- 日志中心。

验收：

- 管理员可创建、删除、封禁、解封用户。
- 管理员可查看和删除全站图片。
- 管理员可设置用户组配额。
- 管理员可管理标签。
- 管理员可管理存储。
- 管理员可配置站点基础信息、SMTP、验证码、主题。

## 5. 第五阶段：审核与 AI

目标：建立图片内容安全能力。

任务：

- 关键词审核。
- 人工审核。
- FastAPI AI 审核服务。
- 大模型审核配置。
- 审核日志。
- 审核队列。

验收：

- 访客上传默认进入审核。
- 普通用户按用户组决定审核策略。
- 管理员可人工通过或拒绝图片。
- 系统可调用 FastAPI 审核接口。
- 系统可配置大模型审核接口。
- 审核日志可追踪。

## 6. 第六阶段：随机图与 API

目标：完善图床对外调用能力。

任务：

- 随机图接口。
- 按标签筛选。
- 按横竖图筛选。
- 按分辨率筛选。
- 按比例筛选。
- 用户 API Token。
- 管理员 API 管理。
- API 限流。

验收：

- 随机图支持标签筛选。
- 随机图支持横图、竖图、方图。
- 随机图支持分辨率匹配。
- 随机图支持返回 redirect、json、markdown、html。
- 用户可创建和删除 API Token。

## 7. 第七阶段：迁移与备份

目标：支持长期运营维护。

任务：

- 图片迁移任务。
- 存储迁移日志。
- 迁移暂停/继续/取消。
- 数据库备份。
- 图片元数据备份。
- 图片文件备份，可选。
- 备份恢复。
- 定时备份。

验收：

- 管理员可从一个存储迁移到另一个存储。
- 迁移支持复制、移动、备份模式。
- 迁移失败可重试。
- 管理员可创建网站备份。
- 管理员可查看备份列表。
- 管理员可从备份恢复。

## 8. 第八阶段：移动端体验完善

目标：让手机端可以完成核心操作。

任务：

- 用户端底部导航。
- 移动端上传体验。
- 移动端图片列表。
- 移动端图片详情。
- 移动端回收站。
- 管理员移动端抽屉菜单。
- 管理员表格转卡片。
- 移动端性能优化。

验收：

- 手机能注册登录。
- 手机能上传图片。
- 手机能查看和删除自己的图片。
- 手机能复制 Markdown / HTML / 原图链接。
- 手机能进入回收站恢复图片。
- 手机能查看配额。
- 管理员手机能审核图片。
- 管理员手机能封禁用户。
- 管理员手机能查看存储状态。

## 9. 第一版总体验收标准

- 用户可以邮箱注册和登录。
- 用户可以上传图片。
- 用户可以管理自己的图片。
- 用户可以复制 Markdown / HTML / 原图链接。
- 管理员可以进入后台。
- 管理员可以管理用户和图片。
- 访客上传可由后台开关控制。
- 图片上传支持哈希去重。
- 图片上传后自动生成 WebP 预览图。
- 用户组配额生效。
- 删除图片进入回收站。
- 标签可创建和筛选。
- 随机图可按标签、方向、分辨率调用。
- 支持本地、R2、OneDrive、Oracle S3、Oracle OCI 原生存储。
- 存储器接管上传、预览、调用。
- 管理员可配置主题、SMTP、验证码、网站基础信息。
- 移动端可完成核心用户操作和基础管理员操作。
- 支持图片迁移。
- 支持网站备份。
