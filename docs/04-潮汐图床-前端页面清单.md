# 潮汐图床前端页面清单

## 1. 视觉方向

潮汐图床前端采用：

- 磨砂玻璃卡片风格。
- 圆润字体。
- 浅色 / 深色主题。
- 管理员后台可切换配色。
- 后台可配置圆角大小。
- 后台可配置卡片模糊程度。
- 支持马卡龙等预设主题。
- 支持根据背景自动取色。

默认字体候选：

- HarmonyOS Sans。
- MiSans。
- Nunito。
- Inter Rounded。
- LXGW WenKai。

## 2. 用户端页面

### 2.1 首页上传页

功能：

- 登录用户上传。
- 访客上传，受后台开关控制。
- 拖拽上传。
- 点击上传。
- 移动端相册/拍照上传。
- 上传进度。
- 上传失败重试。
- 上传成功后显示链接。
- 未登录时展示访客上传策略提示。
- 登录、注册、重置密码通过顶部弹窗进入，不作为首页同级卡片。

显示：

- 当前用户组。
- 今日剩余上传次数。
- 今日剩余容量。
- 单文件限制。
- 是否需要审核。
- 是否需要验证码。

上传结果：

- 原图链接。
- Markdown 链接。
- HTML 链接。
- WebP 预览图链接。
- WebP Markdown 链接。
- WebP HTML 链接。
- 审核状态。
- 哈希复用提示。
- 上传结果缩略图可点击预览。
- Markdown 链接入口打开部署引用弹窗，展示原图 URL、WebP URL、原图 Markdown、WebP Markdown、原图 HTML、WebP HTML。

### 2.2 认证弹窗

功能：

- 邮箱或用户名/密码登录。
- 验证码或 Turnstile。
- 忘记密码入口。
- 注册入口。
- 登录成功后立即保存登录态、关闭弹窗并刷新当前用户信息。

### 2.3 注册弹窗

功能：

- 邮箱注册。
- 邮箱验证码。
- 设置用户名。
- 设置密码。
- 验证码或 Turnstile。

### 2.4 公共图库

功能：

- 未登录可访问。
- 只展示正常公开图片。
- 点击图片打开预览弹窗。
- Markdown 链接入口打开六项部署引用弹窗。
- 按标签筛选。
- 按横图/竖图/方图筛选。
- 复制原图链接。
- 复制 Markdown 链接。

禁止：

- 不展示私有图片。
- 不展示隐藏图片。
- 不展示待审核图片。
- 不展示回收站图片。

### 2.5 我的图片

功能：

- 图片网格。
- WebP 预览图。
- 点击图片打开预览弹窗。
- 详情按钮进入图片详情。
- Markdown 链接入口打开六项部署引用弹窗。
- 按标签筛选。
- 按横图/竖图/方图筛选。
- 按分辨率筛选。
- 按审核状态筛选。
- 复制链接。
- 删除到回收站。

每张卡片显示：

- WebP 预览图。
- 图片名称。
- 尺寸。
- 大小。
- 标签。
- 审核状态。
- 复制按钮。
- 详情按钮。
- 删除按钮。

未登录状态显示登录引导，不请求私有接口。
未登录状态不得在导航中展示我的图片、图片详情、我的回收站、个人资料、API Token 入口。

### 2.6 图片详情

功能：

- 先展示 WebP 预览图。
- 可切换查看原图。
- 编辑标题和描述。
- 编辑标签。
- 复制原图链接。
- 复制 Markdown 链接。
- 复制 HTML 链接。
- 删除到回收站。

显示：

- 原始文件名。
- 文件大小。
- 图片尺寸。
- 横竖图类型。
- 上传时间。
- 审核状态。
- 存储位置。

未登录状态显示登录引导，不请求私有接口。

### 2.7 我的回收站

功能：

- 查看已删除图片。
- 显示剩余保留天数。
- 恢复图片。
- 永久删除图片。

回收站只允许登录用户查看自己的已删除图片，未登录不得查看任何用户回收站。
未登录状态不得在导航中展示回收站入口。

## 2.10 导航可见性

桌面端：

- 顶栏只保留品牌、登录 / 注册、用户状态和管理员后台入口。
- 侧边栏作为主导航。
- 未登录侧边栏只展示首页上传和公共图库。
- 登录后侧边栏展示我的图片、我的回收站、个人资料、API Token。
- 管理员登录后额外展示管理员后台入口。
- 管理员后台入口在顶部和侧边栏同时展示。

移动端：

- 底部导航未登录只展示首页和公共图库。
- 登录后展示图片、回收站、我的。

### 2.8 个人资料

功能：

- 修改用户名。
- 修改密码。
- 修改头像。
- 查看用户组。
- 查看配额。
- 查看 API Token。
- 查看随机图调用示例。

### 2.9 API Token 页面

功能：

- 创建 Token。
- 删除 Token。
- 查看 Token 权限。
- 设置过期时间。

显示调用示例：

- 上传接口。
- 随机图接口。
- AI 接口。

## 3. 管理员后台页面

### 3.1 仪表盘

展示：

- 用户总数。
- 今日注册数。
- 图片总数。
- 今日上传数。
- 存储占用。
- 待审核图片。
- AI 审核统计。
- 最近上传记录。
- 最近封禁记录。
- 存储健康状态。

### 3.2 后台信息架构

后台一级菜单精简为：

- 仪表盘。
- 图片与标签。
- 用户与配额。
- 审核管理。
- 存储与迁移。
- 系统设置。

归并规则：

- 上传管理、随机图管理、外观设置、日志中心、备份恢复归入系统设置。
- 用户组配额归入用户与配额。
- 标签管理归入图片与标签，不再作为一级菜单。
- 图片迁移归入存储与迁移。

### 3.3 用户与配额

功能：

- 创建用户。
- 删除用户。
- 封禁用户。
- 解封用户。
- 修改用户组。
- 查看配额使用。
- 单独覆盖用户配额。
- 查看用户上传图片。
- 创建用户组。
- 设置每日上传数量。
- 设置每日上传容量。
- 设置单文件大小。
- 设置账号总容量。
- 设置是否需要审核。
- 设置是否强制验证码。
- 设置是否允许 API 上传。
- 设置默认存储位置。

### 3.4 图片与标签

功能：

- 查看全部图片。
- 按用户筛选。
- 按标签筛选。
- 按存储筛选。
- 按审核状态筛选。
- 按访客上传筛选。
- 删除图片。
- 审核图片。
- 查看哈希和引用次数。
- 查看原图和预览图存储位置。
- 查看所有标签。
- 搜索标签。
- 禁用标签。
- 删除违规标签。
- 查看标签下图片。

管理员图片卡片显示：

- 上传用户。
- 用户组。
- 存储位置。
- 文件哈希。
- 引用次数。
- 是否访客上传。
- 审核结果。

标签管理说明：

- 标签能力保留，但不作为后台一级菜单。
- 标签策略相关配置归入系统设置的上传设置。
- 标签合并若后端未提供接口，前端不得展示可执行入口。

### 3.5 审核管理

- 审核队列。
- 关键词库。
- AI 审核设置。

### 3.6 存储与迁移

- 测试连接。
- 测试上传。
- 测试删除。
- 默认存储策略。
- 存储路由。

## 4. 移动端支持

### 4.1 断点

- mobile：小于 640px。
- tablet：640px - 1024px。
- desktop：大于 1024px。
- wide：大于 1440px。

### 4.2 用户端移动导航

- 首页。
- 我的图片。
- 随机图。
- 回收站。
- 我的。

### 4.3 管理员移动端

管理员后台移动端采用：

- 顶部管理员入口。
- 抽屉式菜单。
- 表格转卡片列表。
- 筛选项折叠面板。
- 批量操作放到底部操作栏。

### 4.4 移动端验收标准

- 手机能注册登录。
- 手机能上传图片。
- 手机能查看和删除自己的图片。
- 手机能复制 Markdown / HTML / 原图链接。
- 手机能进入回收站恢复图片。
- 手机能查看配额。
- 管理员手机能审核图片。
- 管理员手机能封禁用户。
- 管理员手机能查看存储状态。

### 4.5 移动端性能策略

- 降低毛玻璃 blur 强度。
- 减少大面积 backdrop-filter。
- 避免多层毛玻璃嵌套。
- 背景图自动压缩。
- 图片列表默认加载 WebP 预览图。
