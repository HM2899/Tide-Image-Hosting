# 潮汐图床需求文档

## 1. 用户系统

### 1.1 注册与登录

功能要求：

- 邮箱注册。
- 邮箱验证码验证。
- 用户登录。
- 用户退出。
- 登录失败次数限制。
- 注册、登录可开启验证码或 Cloudflare Turnstile。
- 管理员和普通用户使用同一个登录入口。
- 登录入口支持邮箱或用户名加密码登录。
- 管理员登录后顶部和侧边栏显示管理员后台入口。

### 1.2 用户资料

用户可修改：

- 用户名。
- 密码。
- 头像。

个人中心展示：

- 当前用户组。
- 上传配额。
- 已用容量。
- 今日剩余上传次数。
- 今日剩余上传容量。
- API Token。
- 随机图调用地址。

## 2. 访客上传

后台增加访客上传开关。

配置项：

- 是否允许访客上传。
- 访客上传归属账号。
- 访客上传审核策略。
- 访客上传每日 IP 限制。
- 访客上传单文件大小限制。
- 访客上传允许文件类型。
- 是否启用验证码。
- 是否启用 Cloudflare Turnstile。

业务规则：

- 访客上传关闭时，未登录用户无法上传。
- 访客上传开启时，图片统一归属到指定访客公用账号。
- 数据库额外记录访客 IP、User-Agent、指纹信息。
- 访客上传默认创建审核任务，图片先正常展示，审核拒绝后直接永久删除图片并保留审核记录。

## 3. 图片上传与管理

### 3.1 上传能力

支持：

- 点击上传。
- 拖拽上传。
- 批量上传。
- 移动端相册选择。
- 移动端拍照上传。
- 上传进度。
- 失败重试。
- 文件大小限制。
- 文件类型限制。
- 上传前校验。

- Markdown 链接。

- WebP 预览图链接。
- WebP Markdown 链接。
- WebP HTML 链接。

- 查看图片详情。
- 点击图片直接预览。
- 删除图片到回收站。
- 恢复图片。
- 永久删除图片。
- 编辑标签。
- 复制原图链接。
- 复制 Markdown 链接。
- 复制 HTML 链接。
- 查看并复制原图 URL、WebP URL、原图 Markdown、WebP Markdown、原图 HTML、WebP HTML。

### 3.3 管理员图片管理

管理员可：

- 查看全站图片。
- 按用户筛选。
- 按标签筛选。
- 按存储位置筛选。
- 按审核状态筛选。
- 按访客上传筛选。
- 删除用户图片。
- 审核图片。
- 查看图片所在存储位置。
- 查看哈希、引用次数、对象路径。

## 4. 哈希去重

目标：减少重复图片占用，降低潮汐图床的公益运行成本。

第一版实现：

- SHA-256 精确去重。

上传流程：

```text
用户上传图片
↓
服务端计算 SHA-256
↓
查询 file_objects 是否已有相同 sha256
↓
如果存在，复用已有真实文件
↓
为当前用户创建新的 images 记录
```

删除规则：

- 用户删除图片时，先进入回收站。
- 永久删除用户图片时，只删除该用户的 images 引用。
- 只有 file_object.ref_count = 0 时，才允许删除真实存储文件。

后续增强：

- pHash 感知哈希。
- 相似图片检测。

## 5. WebP 预览图

目标：自动生成 WebP 预览图，用于图片列表、后台管理、详情页预加载，减少原图直出压力。

处理流程：

```text
上传原图
↓
计算哈希
↓
保存或复用原图
↓
读取图片尺寸
↓
生成 WebP 预览图
↓
上传预览图到存储器
↓
写入 storage_objects
```

默认配置：

- 启用 WebP 预览图：是。
- 最大宽度：512px。
- 最大高度：512px。
- WebP 质量：75。
- GIF 预览：取首帧。
- 超大图：生成低清占位图。

前端使用：

- 我的图片列表加载 WebP 预览图。
- 管理后台图片列表加载 WebP 预览图。
- 图片详情页先展示 WebP，再查看原图。
- 公共图库、我的图片和上传结果点击图片时打开预览弹窗。
- Markdown 链接入口打开部署引用弹窗，展示原图和 WebP 两套 URL、Markdown、HTML 引用。
- 随机图接口可选择返回原图或预览图。

## 6. 用户组配额

默认用户组：

- 访客用户。
- 普通用户。
- 可信用户。
- 公益支持者。
- 管理员。

每个用户组可配置：

- 每日上传数量。
- 每日上传总容量。
- 单文件最大大小。
- 账号总存储容量。
- API 每日调用次数。
- 随机图调用次数。
- 是否允许批量上传。
- 是否允许创建标签。
- 是否需要审核。
- 是否强制验证码。
- 默认存储位置。

公益图床推荐策略：

- 访客用户：低额度、强制验证码、强制审核。
- 普通用户：基础配额、默认 AI / 关键词审核。
- 可信用户：更高额度、审核强度降低。
- 公益支持者：更高额度、可分配更优存储策略。
- 管理员：不受普通配额限制。

## 7. 回收站

用户回收站：

- 用户删除图片后进入回收站。
- 回收站图片不再公开访问。
- 用户可恢复图片。
- 用户可永久删除图片。
- 默认保留 30 天。

管理员回收站：

- 查看全站回收站。
- 按用户筛选。
- 按删除时间筛选。
- 按存储位置筛选。
- 恢复图片。
- 永久删除图片。
- 查看删除原因和操作者。

图片状态：

- active：正常。
- trashed：回收站。
- deleted：已永久删除。
- blocked：违规隔离。

用户可：

- 上传时填写标签。
- 图片详情页编辑标签。
- 输入标签后按回车创建。
- 删除自己图片上的标签。
- 按标签筛选自己的图片。

管理员可：

- 查看所有标签。
- 按标签搜索图片。
- 合并重复标签。
- 删除违规标签。
- 禁用标签。
- 查看标签使用次数。

后台可配置：

- 是否允许用户创建新标签。
- 新标签是否需要审核。
- 每张图片最大标签数。
- 标签最大长度。
- 标签敏感词过滤。

## 9. 随机图接口

支持按标签：

```text
GET /random
GET /random?tag=风景
GET /random?tags=风景,壁纸
GET /random?tags=风景,壁纸&mode=all
```

支持按方向：

```text
GET /random?orientation=landscape
GET /random?orientation=portrait
GET /random?orientation=square
```

支持按分辨率：

```text
GET /random?min_width=1920&min_height=1080
GET /random?width=1920&height=1080&match=exact
GET /random?width=1920&height=1080&match=near&tolerance=100
GET /random?ratio=16:9
```

支持返回格式：

```text
GET /random?type=redirect
GET /random?type=json
GET /random?type=markdown
GET /random?type=html
GET /random?image=preview
GET /random?image=original
```

## 10. 审核系统

第一版实现：

- 关键词审核。
- AI 审核接口对接。
- 大模型审核。
- 人工审核。

可按来源配置：

- 访客上传：强制审核。
- 普通用户上传：按用户组配置。
- 管理员上传：默认跳过审核。
- 审核后置执行，上传成功后先展示图片；审核拒绝后永久删除图片和存储对象，保留审核结果与日志。

审核对接设置：

- 审核服务开关。
- 审核服务类型：内置审核 / FastAPI AI 审核 / 第三方接口 / 大模型审核。
- 接口地址。
- API Token。
- 请求超时时间。
- 失败重试次数。
- 失败策略：转人工 / 拒绝 / 放行。
- 是否启用图片 AI 审核。
- 是否启用 OCR。
- 是否启用关键词审核。
- 是否启用文件名审核。

大模型支持：

- OpenAI Compatible。
- Azure OpenAI。
- Ollama。
- Gemini，可选。
- Anthropic，可选。
- 其他视觉模型接口。

## 11. 图片迁移

管理员可将已有图片从一个存储位置迁移到另一个存储位置。

迁移范围：

- 全部图片。
- 按用户迁移。
- 按用户组迁移。
- 按标签迁移。
- 按存储位置迁移。
- 按上传时间迁移。
- 按图片状态迁移。
- 只迁移原图。
- 只迁移 WebP 预览图。
- 原图和预览图一起迁移。

迁移模式：

- 复制模式：复制到新存储，旧文件保留。
- 移动模式：复制成功后删除旧存储文件。
- 备份模式：新存储作为副本，旧存储仍为主存储。

任务能力：

- 断点续传。
- 失败重试。
- 迁移日志。
- 迁移进度。
- 并发数量限制。
- 失败文件列表。
- 完成后校验哈希。
- 暂停/继续/取消。

## 13. 安全要求

登录安全：

- 密码使用 Argon2 加密。
- 邮箱验证码。
- 登录失败限制。
- IP 限流。
- Session / Token 过期。
- 管理员关键操作二次确认。

上传安全：

- 文件类型白名单。
- 文件大小限制。
- MIME 检测。
- 图片解码验证。
- 禁止危险格式作为公开直出。
- 去除 EXIF，可配置。
- 访客上传强制验证码。

API 安全：

- API Token 可单独生成和吊销。
- Token 权限范围。
- Token 过期时间。
- IP 白名单，可选。
- 调用频率限制。
